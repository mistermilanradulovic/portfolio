# Portfolio

A personal portfolio built with Vite, React, TypeScript, shadcn/ui, and Tailwind CSS.

## Getting started

1. Install dependencies:
   ```sh
   npm i
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```
3. Build for production:
   ```sh
   npm run build
   ```

## Tech stack

- Vite
- React (TypeScript)
- shadcn/ui
- Tailwind CSS
